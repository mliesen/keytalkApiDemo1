# KeyTalk logvalues

Logga keytalk värden och skriv till textfil.
Loggning sker med prenumeration, varje ändring skrivs ner  


Framtida - flytttal kan ges en floatRes vid prenumerationsstart


# Installation
i root mappen:

```
npm install
```

Köra:

Ställ dig i mappen med config.json. Kör
```
node ./dist/index.js
```
